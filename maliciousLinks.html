What is a URL?
protocolhostnamepathname
https://www.leagueoflegends.com/en-gb/champions/katarina/

A URL is a link used to access content on the internet and is simply a string of characters, as shown in the image, which is required to access any content using a browser such as chrome on your computer

The URL can usually be seen at the top of the page, for example when you open a page on the League of Legends website, as shown here

The URL usually starts with http or https, followed by ://, then some understandable string like www.leagueoflegends.com, which you can obviously understand is a link to League of Legends, and then some additional strings after that

ChildSecurity
ABOUT
This is a web application to help train children to be aware of cyber security and privacy while playing online games. The application will describe typical cyber security and privacy risks in detail and then express the damage caused by these risks through hack or cyber attack operations, giving you a realistic experience of a cyber attack.
The goal of this application is to light up the skills tree, which contains 4 educational topics: Malicious Links, Cyberbullying, Sensitive Information and Password Security. Each topic is a learning process, from introducing the basic concepts to experiencing the dangers and quizzes after learning the topic.
